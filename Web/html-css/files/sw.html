EPISODE IV
Les priorités entre sélecteurs CSS

Dans l'exemple ci-dessous 8 règles entrent en concurrence.

C'est finalement la couleur orange qui est appliquée.

Voici les poids de chaque sélecteur. Cyan et orange ont le même poids. Du coup c'est le dernier qui est sélectionné

h1 {color :green;} /*0001*/
.fruit {color:red;}/*0010*/
h1.fruit{color:yellow;}/*0011*/
div h1.fruit span {color:blue;}/*0012*/
.container .fruit span {color:white;}/*0021*/
div.container .fruit span {color:cyan;}/*0022*/
div.container .fruit span {color:orange;}/*0022*/
.container .fruit span {color:purple;}/*0021*/

N'hésitez pas modifier les sélecteurs ou changer les lignes de places pour observer les priorités changer.
